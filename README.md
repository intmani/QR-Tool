## QR Tool


**A CLI tool to generate QR codes and scan QR codes from image or camera.**



## Features

**- Generate QR (PNG)**

**- Scan QR from image**

**- Scan QR from camera**


## Setup

```
python -m venv venv
venv\\Scripts\\activate
pip install -r requirements.txt
```


## Run

```
python main.py
```



**Notes**

* **Image paths with quotes are supported**
* **If camera index 0 doesn't work, try 1 or 2**
